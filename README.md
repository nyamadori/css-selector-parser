css-selector-parser
===================

This is the parser for CSS Selector that [CSS Selector Level 4](http://dev.w3.org/csswg/selectors-4/) compliant.
It is created by [PEG.js](http://pegjs.org/).

Install
-------

```
$ npm install
```

Development
-----------

### PEG.js (*.pegjs) compile & Test

```
$ npm install # (compile)
$ npm test
```
